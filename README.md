## 🧠 Who I Am

I'm Dharam — a solo founder building AI infrastructure from Gandhidham, Kutch, Gujarat.

I don't use no-code tools. I don't outsource. I don't build SaaS wrappers around OpenAI.

I build **full stacks** — reasoning engines, spatial OSes, visual app builders, research agents, gamified platforms — all local-first, all free-forever at core, all owned end-to-end.

My operating philosophy: **sovereign tech, Bharat-first, ship daily.**

> "If it requires a US credit card and a cloud subscription to think, it's not intelligence — it's a rental."

## 🏗️ The Daxini Stack

The Daxini Stack is a suite of interconnected local-first AI systems. Every layer runs on your machine.

```
┌─────────────────────────────────────────────────────┐
│                   DAXINI STACK                      │
├──────────────┬──────────────┬───────────────────────┤
│  Zayvora     │     Nex      │      Orchade           │
│  Reasoning   │  Research    │   Farming Game /       │
│  Engine      │  Agent       │   Social OS            │
├──────────────┴──────────────┴───────────────────────┤
│              StudyOS  ·  Orchade                    │
├─────────────────────────────────────────────────────┤
│           daxini.xyz  (Mothership Portal)           │
├─────────────────────────────────────────────────────┤
```

### Core Modules

| Module | Repo | Description |
|--------|------|-------------|

| 🌌 **Daxini.space** | [daxini.space](https://daxini.space) | Gesture-based Spatial OS PWA with node registry and iframe trust chain |

## 🧩 Philosophy

```
Local-first  →  No cloud dependency by default
Sovereign    →  Own your compute, own your data
Bharat-first →  Built for India's infrastructure realities
Free-forever →  Core access never paywalled
Zero npm     →  Vanilla JS for core products where possible
Ship daily   →  Building in public, always
```

> I'm not building for YC. I'm not building to exit.  
> I'm building infrastructure that works when the internet is slow,  
> the cloud is expensive, and the data is yours.
